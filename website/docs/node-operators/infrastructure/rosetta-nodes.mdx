# Rosetta Nodes

o1Labs maintains Rosetta API nodes for both mainnet and devnet networks. Rosetta
is a standard interface for blockchain integration, primarily used by exchanges
and custodians for account balance tracking and transaction construction.

## Available Rosetta Nodes

The following Rosetta nodes are officially maintained by o1Labs:

- **Devnet**:
  [`https://devnet-rosetta.gcp.o1test.net/`](https://devnet-rosetta.gcp.o1test.net/network/status)
- **Mainnet**:
  [`https://mainnet-rosetta.gcp.o1test.net/`](https://mainnet-rosetta.gcp.o1test.net/)

## Implementation Status

:::info OCaml Implementation

The mina-rust codebase does not currently have a Rosetta interface
implementation. The available Rosetta nodes run the OCaml implementation from
the Mina Protocol codebase.

## Use Cases

Rosetta nodes are primarily used for:

- **Exchange integration**: Standardized API for cryptocurrency exchanges
- **Account tracking**: Monitor balances and transaction history
- **Transaction construction**: Build and submit transactions
- **Network information**: Query network status and configuration
