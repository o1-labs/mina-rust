# Benchmarks

Performance benchmarks for Mina components using
[Criterion](https://github.com/bheisler/criterion.rs).

## Building benchmarks

Build all benchmarks without running them:

<CodeBlock language="bash" title="scripts/benchmarks/build-benchmarks.sh">
  {BuildBenchmarks}
</CodeBlock>

## Running benchmarks

Run all benchmarks:

<CodeBlock language="bash" title="scripts/benchmarks/run-all-benchmarks.sh">
  {RunAllBenchmarks}
</CodeBlock>

## Ledger

### Database benchmarks

Benchmarks for ledger database operations including account generation and
merkle root computation. Tests performance with 1,000, 10,000, and 120,000
accounts.

Run the database benchmark:

<CodeBlock language="bash" title="scripts/benchmarks/run-database-benchmark.sh">
  {RunDatabaseBenchmark}
</CodeBlock>

Benchmark groups:

- **account_generation**: Measures time to generate random accounts and insert
  them into the database
- **merkle_root_computation**: Measures time to compute the merkle root for a
  populated database

Results are saved to `target/criterion/` with detailed HTML reports.
